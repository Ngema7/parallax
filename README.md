# Parallax Website

A simple parallax scrolling website built with **HTML** and **CSS**, demonstrating the parallax effect where background images move slower than the foreground content to create depth.

## Features

- Multiple sections with parallax background images  
- Overlay text on images  
- Responsive design  
- Clean and minimal HTML/CSS structure
